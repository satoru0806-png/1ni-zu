# 夢ナビ Watch 設計図

## コンセプト

**「手首から人生を整える」**

スマートウォッチから声で入力し、AIコーチが朝・昼・夜に語りかける。
全ての記録は「人生の日記」として蓄積され、いつでも振り返れる財産になる。

---

## 対象デバイス

| デバイス | OS | 開発言語 | 画面 |
|---------|-----|---------|------|
| Apple Watch | watchOS | Swift / SwiftUI | 丸角四角 396x484px |
| Garmin Forerunner 55 | Connect IQ | Monkey C | 円形 208x208px |
| Wear OS (Pixel Watch等) | Wear OS | Kotlin / Compose | 円形 384x384px |

**Phase 1**: Apple Watch（市場最大・開発環境が整っている）
**Phase 2**: Wear OS
**Phase 3**: Garmin（Connect IQ制約あり）

## 通信アーキテクチャ

```
┌──────────┐   BLE    ┌──────────┐   HTTPS   ┌──────────┐
│          │ ←-----→  │          │ ←------→  │          │
│  Watch   │          │  iPhone  │           │  夢ナビ   │
│  App     │          │  /Android│           │  Server   │
│          │          │  (中継)  │           │ (Vercel)  │
└──────────┘          └──────────┘           └──────────┘
     │                      │                      │
     │ 音声認識              │ Watch                │ API
     │ (オンデバイス)         │ Connectivity         │ /api/daylog
     │                      │                      │ /api/dreams
     │ ローカル              │ 中継                  │ /api/history
     │ キャッシュ             │ + 通知               │ /api/push
     │                      │ スケジュール           │ /api/cron
```

### データフロー

```
1. 音声入力（Watch上でオンデバイス認識）
   ↓
2. テキストをiPhone経由でサーバーに送信
   ↓
3. AI整形（Claude API）
   - フィラー除去
   - タスク抽出
   - 感情分析
   - スコア提案
   ↓
4. 結果をWatchに返却・表示
   ↓
5. ユーザー確認 → DB保存
```

## 技術仕様

### Apple Watch (watchOS)

| 項目 | 仕様 |
|------|------|
| 最小OS | watchOS 10 |
| 言語 | Swift 6 / SwiftUI |
| 通信 | Watch Connectivity Framework |
| 音声 | SFSpeechRecognizer（オンデバイス） |
| 通知 | UserNotifications + WatchKit |
| ストレージ | Core Data（ローカルキャッシュ） |
| コンプリケーション | WidgetKit |

### Wear OS

| 項目 | 仕様 |
|------|------|
| 最小OS | Wear OS 4 |
| 言語 | Kotlin |
| UI | Compose for Wear OS |
| 通信 | Data Layer API |
| 音声 | SpeechRecognizer |
| タイル | Tiles API |

### Garmin Connect IQ

| 項目 | 仕様 |
|------|------|
| SDK | Connect IQ 3.2.5+ |
| 言語 | Monkey C |
| 通信 | Communications.makeWebRequest() |
| 制約 | 音声認識なし → テンプレート選択式 |
| Widget | Forerunner 55: 208x208px |

## 日記機能設計（人生の財産）

### データモデル拡張

```
model LifeJournal {
  id          Int      @id @default(autoincrement())
  date        String   @unique  // YYYY-MM-DD
  year        Int                // 年別検索用
  month       Int                // 月別検索用

  // 朝の記録
  morningMood    String?         // 朝の気分
  mits           String?         // MIT（JSON）

  // 昼の記録
  progressNote   String?         // 進捗メモ

  // 夜の記録
  doneNote       String?         // できたこと
  gratitudeNote  String?         // 感謝
  tomorrowPlan   String?         // 明日の計画
  eveningMood    String?         // 夜の気分

  // スコア
  relationshipScore Int @default(50)
  moneyScore       Int @default(50)
  workScore        Int @default(50)
  healthScore      Int @default(50)

  // AI生成
  aiSummary      String?         // AIによる1日の要約
  aiInsight      String?         // AIからの気づき
  memoRaw        String?         // 生の音声テキスト
  sentiment      String?         // 感情分析結果

  // メタ
  inputMethod    String?         // "voice" | "text" | "watch"
  createdAt      DateTime @default(now())
  updatedAt      DateTime @updatedAt
}
```

### 振り返り機能

| 期間 | 表示内容 |
|------|---------|
| 1日 | その日の全記録 + AIの一言 |
| 1週間 | スコアグラフ + 週の振り返りAI要約 |
| 1ヶ月 | カレンダーヒートマップ + 月の成長ポイント |
| 1年 | 年間ハイライト + 夢への進捗 |
| 「1年前の今日」 | タイムカプセル（去年の自分からの手紙） |

## まとめ

夢ナビ Watchは「**声で人生を記録し、時計が人生のコーチになる**」ツール。

Typelessが「どこでも使える音声入力」なら、
夢ナビは「**人生の夢に向かって毎日導いてくれる音声コーチ**」。

全ての記録は日記として蓄積され、
1年後、5年後、10年後に振り返った時、
**自分の成長が見える人生の財産**になる。
